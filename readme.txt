Lightweight plugin to integrate Social Share Icons.

== Description ==
Easy and light plugin to include social share icons in your project.
Easy to set up and fast! No unneeded overload.
Supported Social Networks:
    - Twitter
    - Google+
    - Facebook
    - Pinterest
    - Linkedin
    - Whatsapp

Features:
 * Activation / Deactivation of Social Networks
 * Drag and Drop Sorting
 * 3 sizes to choose (Small / Medium / Large)
 * Colour Picker for Icons
 * Choose where to Display them (Posts / Pages / CPTs)
 * 4 Places Available:
    * Bellow Post Title
    * Floating left (Sticky)
    * After Post Content
    * Inside Featured Image
 * Shortcode ( [social_share] )

== Installation ==
1. Download and place it under /wp-content/plugins
2. Activate it.
3. Go to Setting Pages and set up as you please.


== Frequently Asked Questions ==

= Is it Free? =
100%

= Is it Fast? =
Probably this is the lightest plugin you can find out there.

= Is it Configurable? =
Definitely yes, you can chose where to place it, which social networks to show, sizes, colours and post types.

= Is a shortcode available? =
[social_share]

= How does Whatsapp share icon works? =
It works only when the user is on a mobile or table device.
